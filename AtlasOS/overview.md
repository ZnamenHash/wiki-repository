# AtlasOS

AtlasOS — это открытый проект, предоставляющий удобные скрипты и инструменты для глубокой оптимизации Windows без жёсткой кастомизации ISO и потери совместимости с обновлениями и сервисами. 

## Ключевые возможности AtlasOS
- Удаление телеметрии Microsoft и навязчивых служб сбора данных
- Минимизация фоновых процессов для высвобождения RAM и CPU
- Отключение рекламы, рекомендаций и аналитики Windows
- Конфигурируемые настройки безопасности (Windows Defender, SmartScreen, обновления)
- Полная совместимость с магазинами, играми и античитами
- Чистое, реверсивное внедрение оптимизаций: можно обойтись без перепрошивки системы

## Playbook и Wizard
Atlas Playbook — пакет сценариев для TrustedInstaller, позволяющий массово применять или возвратить твики. Atlas Wizard — отдельное приложение для удобной настройки твиков по категориям.

Подробнее: [AtlasOS docs](https://docs.atlasos.net/)

## Установка
1. Внимательно изучите инструкции на [официальном сайте AtlasOS](https://atlasos.net/)
2. Не рекомендуется использовать на кастомных ISO, только на чистых официальных сборках Windows 10/11
3. Перед применением сделайте резервную копию данных
4. После установки используйте Atlas Wizard для доработки системы под себя

## Преимущества и риски
- Прирост FPS в играх и быстрая загрузка
- Свободная RAM (до 1,5 ГБ)
- Ускорение отклика и уменьшение задержек
- Высокая репутация — используется киберспортсменами и энтузиастами
- Нет откатов из коробки: возврат только через переустановку Windows
