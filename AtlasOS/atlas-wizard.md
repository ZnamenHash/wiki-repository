# Atlas Wizard

Atlas Wizard — это графический интерфейс и мастер для применения твиков и оптимизаций AtlasOS на Windows 10/11 без необходимости ручного запуска скриптов.

## Ключевые возможности
- Модульное управление оптимизациями (можно включать/выключать отдельные твики)
- Категоризация твиков по разделам: безопасность, производительность, приватность, интерфейс
- Загрузка и применение последних версий Playbook из облака
- Проверка совместимости с текущей версией Windows
- Автоматизация отката изменений (если возможно)

## Как пользоваться Atlas Wizard
1. Скачайте Atlas Wizard с [официального сайта AtlasOS](https://atlasos.net/download)
2. Запустите от имени администратора
3. Следуйте пунктам мастера для выбора оптимизаций
4. Сохраняйте или экспортируйте профиль выбранных твиков
5. Применяйте изменения: большинство твиков применяются сразу без перезагрузки

## Примеры твиков, доступных в Wizard
- Отключение диагностических служб (DiagTrack и др.)
- Отключение магазинов UWP
- Включение/отключение Windows Defender
- Отключение драйвера Xbox
- Отключение анимаций/визуальных эффектов
- Управление обновлениями Windows

## Особенности
- Интерфейс интуитивно понятен: все опции снабжены описаниями
- Не требует глубоких знаний Windows
- Позволяет создавать индивидуальные профили для разных задач (например, «максимальная производительность», «максимальная совместимость»)
