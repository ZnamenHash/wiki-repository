# Что изменяет Atlas Playbook

Atlas Playbook вносит множество изменений в систему Windows для улучшения производительности, приватности и удобства использования. Здесь представлен подробный список того, что именно меняется.

## Компоненты, которые удаляются навсегда

### Smart App Control
- Полностью отключается при установке Atlas
- Не может быть восстановлен без переустановки Windows
- Отключён, так как отправляет большое количество данных в Microsoft
- Требует включения опциональной диагностики Windows
- Вызывает серьёзные замедления при отключённом Defender

## Службы и компоненты, которые отключаются

### Телеметрия и диагностика
- **DiagTrack** (Connected User Experiences and Telemetry) — основная служба сбора телеметрии
- **Windows Error Reporting** — отправка отчётов об ошибках
- **CEIP** (Customer Experience Improvement Program) — программа улучшения качества обслуживания
- **Diagnostic Policy Service** — служба диагностической политики
- **Diagnostic Execution Service** — выполнение диагностических действий

### Синхронизация и облачные сервисы
- Синхронизация с учётной записью Microsoft (настройки, пароли, темы)
- OneDrive (удаляется полностью, если не настроен)
- Облачная синхронизация буфера обмена
- История активности Windows Timeline

### Реклама и рекомендации
- Реклама в меню Пуск
- Реклама на экране блокировки
- Советы и рекомендации Windows
- Предложения приложений из Microsoft Store
- Windows Tips (устаревшее приложение)

### Контекстные меню и интерфейс
- Редко используемые пункты контекстного меню
- Лишние элементы панели задач
- Windows Widgets (виджеты на рабочем столе)

## Приложения, удаляемые по умолчанию

### Системные приложения Windows
- **Internet Explorer** (только Windows 10) — можно восстановить как дополнительную функцию
- **OneDrive** — если уже настроен, не удаляется (защита от потери данных)
- **Microsoft Edge** (опционально) — пользователь может выбрать, оставить или удалить
  - Можно переустановить через скрипт `Remove Edge.cmd`
- **Steps Recorder** (устаревший) — можно восстановить
- **Legacy Windows Media Player** — заменён современным проигрывателем
- **Maths Recognizer** (распознаватель математических формул)

## Настройки реестра и групповых политик

### Приватность
- Отключение геолокации и отслеживания местоположения
- Блокировка доменов телеметрии Microsoft через hosts-файл
- Отключение фоновой камеры и микрофона
- Отключение рекламного ID
- Отключение доступа приложений к диагностическим данным

### Производительность
- Отключение фоновых приложений UWP
- Оптимизация параметров TCP/IP и сетевого стека
- Настройка QoS (Quality of Service)
- Изменение плана энергопотребления на максимальную производительность
- Отключение индексирования поиска (опционально)

### Игровые оптимизации
- Отключение **Xbox Game Bar**
- Отключение **Xbox DVR** (запись игр)
- Отключение **Game Mode** (опционально)
- Удаление Xbox Services (опционально)

### Сеть
- Отключение **Teredo** (IPv6-туннелирование)
- Отключение **ISATAP**
- Опциональное отключение IPv6
- Оптимизация параметров сетевой карты

## Запланированные задачи

Удаляются или отключаются следующие задачи Task Scheduler:
- Задачи телеметрии Microsoft
- Задачи Customer Experience Improvement Program
- Задачи диагностики системы
- Автоматическая дефрагментация (для SSD)
- Задачи обслуживания Windows
- Задачи облачной синхронизации

## Опциональные компоненты безопасности

Атлас позволяет пользователю выбирать, что оставить:

### Windows Defender (опционально)
- Можно оставить включённым для защиты
- Можно полностью отключить для максимальной производительности
- SmartScreen также настраивается отдельно

### Windows Update
- Автоматические обновления можно отключить
- Безопасность: обновления безопасности можно устанавливать вручную
- Atlas блокирует проблемные feature updates автоматически

### CPU Mitigations (уязвимости процессора)
- **Spectre и Meltdown** — митигации отключены для повышения производительности
- Риск минимален для обычных пользователей, но стоит учитывать

### Другие опции безопасности
- **User Account Control (UAC)** — можно настроить
- **Core Isolation** — технологии изоляции ядра
- **VBS (Virtualization Based Security)** — можно отключить

## Важные замечания

⚠️ **Постоянные изменения:**
- Smart App Control невозможно восстановить без переустановки Windows
- Некоторые UWP-приложения могут работать нестабильно после удаления зависимостей

⚠️ **Восстановление функций:**
- Большинство изменений обратимы через ручные настройки
- Atlas Wizard позволяет включать/выключать некоторые твики
- Полный откат возможен только через переустановку Windows

⚠️ **Обновления Windows:**
- Atlas сохраняет совместимость с Windows Update
- Feature updates блокируются автоматически, если они проблемные
- Механизмы сохранения твиков предотвращают откат изменений при обновлениях
- Не требуется повторное применение Playbook после обновлений безопасности
