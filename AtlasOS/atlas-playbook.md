# Atlas Playbook

Atlas Playbook — это основной пакет автоматизированных скриптов для применения оптимизаций AtlasOS. Playbook использует специальный формат `.apbx` и запускается через AME Wizard (собственный инструмент установки).

## Что делает Playbook

### Оптимизация производительности
- Отключение фоновых служб Windows (telemetry, диагностика, отчёты об ошибках)
- Настройка планировщика задач: удаление ненужных задач
- Оптимизация сетевого стека (TCP/IP параметры)
- Настройка энергопотребления для максимальной производительности
- Отключение Xbox Game Bar, DVR и других игровых оверлеев

### Конфиденциальность
- Блокировка телеметрии Microsoft на уровне хостов и реестра
- Отключение Cortana, поиска Bing в меню Пуск
- Удаление OneDrive интеграции
- Отключение рекламы в меню Пуск и на экране блокировки
- Минимизация отправки диагностических данных

### Интерфейс и UX
- Удаление bloatware приложений (Candy Crush, TikTok и т.д.)
- Оптимизация проводника Windows
- Отключение анимаций и визуальных эффектов (опционально)
- Настройка контекстного меню

## Как применять Playbook

1. Скачайте последнюю версию Atlas Playbook с [GitHub Releases](https://github.com/Atlas-OS/Atlas/releases)
2. Установите AME Wizard (входит в комплект)
3. Запустите AME Wizard от имени администратора
4. Выберите скачанный `.apbx` файл
5. Следуйте инструкциям мастера установки
6. После установки система перезагрузится

## Важные замечания

⚠️ **Перед применением:**
- Создайте точку восстановления системы
- Сделайте резервную копию важных данных
- Убедитесь, что используете чистую официальную Windows 10/11
- Не применяйте на кастомных сборках

⚠️ **После применения:**
- Некоторые функции Windows могут быть недоступны (Cortana, Xbox Live и др.)
- Windows Update будет работать, но с ограничениями
- Возможны проблемы с некоторыми UWP приложениями
- Откат возможен только через переустановку Windows

## Настройки Playbook

Playbook предлагает несколько опций при установке:
- **Уровень оптимизации**: Базовый / Продвинутый / Максимальный
- **Безопасность**: Включить/отключить Windows Defender
- **Обновления**: Настроить поведение Windows Update
- **Совместимость**: Оставить компоненты для игр с античитами

## Компоненты, которые удаляются/отключаются

- Windows Defender (опционально)
- Телеметрия и диагностика
- Cortana
- OneDrive
- Xbox services (опционально)
- Windows Error Reporting
- Scheduled Tasks (выборочно)
- Различные фоновые службы
