# Что изменяет GTweak

GTweak — один из самых детальных твикеров для Windows. Вот полный разбор изменений и возможностей, основанный на исходниках и описаниях из репозитория:

## Безопасность и приватность
- Отключение Windows Defender, SmartScreen и антивирусных модулей
- Деактивация User Account Control (UAC)
- Отключение/удаление встроенного защитника Smart Screen, антивируса, виртуализации VBS
- Отключение служб шпионажа, логирования и кейлоггеров (Windows, NVIDIA)
- Отключение сбора данных в драйверах и сторонних службах (метрики NVIDIA)
- Отключение сбора телеметрии, анализа событий, отслеживания пользователя
- Блокировка теневых доменов Microsoft через hosts

## Производительность и система
- Отключение неиспользуемых служб и компонентов
- Отключение службы обновлений, удаление временных файлов обновлений
- Отключение автозагрузки и кэшей
- Деактивация дефрагментации (для SSD)
- Отключение индексации поиска
- Отключение излишних сетевых (IPv6, Teredo, ISATAP) и диагностических протоколов
- Корректировка звуковой схемы, улучшение отклика Realtek Audio
- Оптимизация работы интерфейса, удаление шайбы, скрытые иконки
- Включение/отключение классического меню Пуск

## Деинсталляция и чистка
- Удаление OneDrive, Edge (включая папки и остаточные данные)
- Удаление UWP и стандартных приложений Windows 10/11
- Удаление Copilot, Cortana, Recall, AI-ассистентов
- Чистка автозагрузки, оптимизация Startup
- Настройка тем и интерфейса

## Дополнительные твики
- Управление питанием и оптимизация для ноутбуков
- Отключение рекламы, баннеров SCOOBE
- Отключение ключевых уведомлений, советов и предложений
- Переключение планов электропитания
- Включение детального BSOD, Verbose Logon messages, поддержка NumLock
- Отключение акселерации мыши, Sticky Keys

## Какие задачи выполняются скриптами GTweak:
- Изменение/восстановление реестра для всех твиков
- Создание/отмена точек восстановления
- Контроль отката твиков (бекупы делаются автоматически)
- Есть интерфейс выбора каждого твика
