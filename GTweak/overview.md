# GTweak

GTweak — это бесплатный инструмент для комплексной настройки и ускорения Windows 10/11. Проект нацелен на быструю приватизацию, удобство использования и устранение типичных лагов без удаления важных сервисов системы.

## Ключевые возможности
- Отключение телеметрии и служб Microsoft
- Оптимизация сети (TCP/IP, QoS)
- Отключение рекламы и уведомлений
- Чистка автозагрузки и памяти
- Быстрые твики для геймеров: отключение GameDVR, Xbox Game Bar
- Полная секция настройки приватности: отключение геолокации, «умного» поиска, истории активности
- Бэкап важных твиков
- Легкое восстановление настроек до дефолта

## Как пользоваться
1. Скачайте последнюю версию GTweak с [GitHub](https://github.com/Greedeks/GTweak)
2. Запустите скрипт от администратора (PowerShell или .exe)
3. Выберите нужные твики/настройки и примените их
4. Система предложит перезагрузку (необязательно — часть твиков работает без неё)

## Разделы твиков
- **Системные**: управление службами, настройка обновлений, дефрагментация
- **Приватность**: отключение сборщиков данных, рекламных модулей
- **Оптимизация интерфейса**: удаление анимаций, Classic меню Пуск
- **Сеть**: ускорение пинга и скорости

## Особенности
- Не затрагивает лицензию или активацию Windows
- Имеет открытый исходный код и простую структуру
- Подходит для ноутбуков и ПК
