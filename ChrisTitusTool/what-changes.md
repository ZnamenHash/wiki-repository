# Что изменяет ChrisTitusTool (WinUtil)

ChrisTitusTool (WinUtil) вносит комплексные изменения и твики для оптимизации, чистки и настройки системы Windows 10/11. Вот подробный разбор изменений, которые может делать данный инструмент.

## Debloat/Удаление лишнего
- Деинсталляция Microsoft Edge (по желанию)
- Удаление OneDrive
- Удаление навязчивых и малоиспользуемых UWP-приложений (Mail, Calendar, Groove, Xbox, People, Skype и др.)
- Очистка временных файлов, кэшей приложений, Windows Update Cache
- Полное удаление или отключение Copilot, Cortana, Recall

## Отключение и настройка служб
- Отключение DiagTrack (Телеметрия)
- Отключение WiFi Sense
- Отключение GameDVR
- Отключение Activity History
- Отключение Storage Sense
- Отключение и удаление Powershell 7 Telemetry
- Отключение Homegroup
- Отключение Location Tracking (службы определения местоположения)
- Выборочные задачи Task Scheduler (Diagnostica, Customer Experience, облачные службы)
- Остановка и освобождение памяти служб, не критичных для работы
- Рекомендуется вручную активировать Ultimate Performance Plan (есть специальный пункт)

## Tweaks/Реестр
- Включение классического контекстного меню (Classic Right-Click Menu)
- Включение/отключение Bing Search в меню Пуск
- Отключение рекламы
- Принудительная тёмная тема (или стандарт)
- Отключение визуальных эффектов для прироста FPS/отклика
- Показ расширений файлов и скрытых файлов
- Отключение автозагрузки ненужных утилит
- Отключение автозапуска приложений/служб после обновлений
- Отключение Sticky Keys и фильтрации клавиш

## Приватность и безопасность
- Отключение Consumer Features
- Отключение фоновых приложений UWP
- Отключение или настройка Windows Defender
- Отключение автоматической отправки ошибок
- Отключение диагностических данных для PowerShell
- Отключение доступа приложений к микрофону и камере (в реестре)
- Отключение использования устройства в целях аналитики
- DNS-твики для повышения приватности (AdGuard, Quad9, OpenDNS)

## Оптимизация сети и системы
- Отключение IPv6 (опционально)
- Применение твиков TCP/IP и QoS
- Отключение Teredo
- Перевод сетевого стека в режим IPv4 по умолчанию
- Очистка DNS Cache и ускорение разрешения имён

## Пакетная установка программ
- Bulk установка через winget, Chocolatey, Scoop (браузеры, VS Code, Steam, утилиты)

## Отмена/Undo
- Любой твик может быть отменён через "Undo Tweaks"
- Создаётся точка восстановления перед изменениями
- Весь функционал работает через GUI или PowerShell-режим


## Важные предупреждения

- Некоторые твики могут сломать часть интеграций или обновлений
- Требуется ручная проверка после серьёзных изменений интерфейса или сети
- Не рекомендуется удалять все Store Apps сразу на основных машинах
- После агрессивных твиков возможен откат только через восстановление
